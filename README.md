# NotiBot-Telegram
NotiBot es un bot de Telegram que obtiene noticias de RSS públicas y las envía a canales/grupos en horarios fijos. Desarrollado en Node.js,respeta límite de 5 noticias/día y evita repetir titulares.Gratuito gracias a API de Telegram y hosting gratuito (Render, Cyclic).Ideal para comunidades que quieran titulares relevantes sin intervención manual.
